# Changelog

## [2.5.0]

### New Features
- Split Screen - Support horizontal/vertical split, operate multiple terminals simultaneously
- Command Snippets - Save frequently used commands, support iCloud sync
- Bug fixes and stability improvements



## [2.4.0]

### New Features
- Jump Host Support - Connect to a target host via an intermediate server. Perfect for multi-hop network environments.
- SOCKS5 Proxy - Now supports dynamic port forwarding. Create a local proxy server with just one click.
- Custom Host Icons - Use built-in icons or image URLs to make your connection list easier to identify.
