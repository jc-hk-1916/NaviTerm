# NaviTerm

Язык / Language: Русский | [English](README.md) | [中文](README.zh.md) | [Español](README.es.md) | [Français](README.fr.md) | [Deutsch](README.de.md) | [日本語](README.ja.md)

Профессиональный SSH-клиент с AI-помощником для macOS, iPhone и iPad. Объединяет SSH-терминал, SFTP, переадресацию портов и автоматизацию. Все для удаленного администрирования в одном приложении. Поддержка iCloud и фонового режима.

---

## Ключевые особенности

### AI-помощник
- Встроенная AI-панель (OpenAI, Claude, Gemini)
- Анализ вывода, исправление ошибок, оптимизация скриптов и проверка безопасности в один клик
- Управление диалогами, настройка шаблонов подсказок
- Умные предложения команд для повышения эффективности

### Автоматизация задач
- Планировщик для автоматического выполнения рутинных задач
- Визуальный редактор скриптов без необходимости кодинга
- Готовые шаблоны: мониторинг, анализ логов, развертывание
- Пакетные операции на нескольких серверах одним кликом

### Фоновый режим iOS + iCloud
- SSH, SFTP и переадресация портов работают в фоне без разрывов
- Мгновенная синхронизация настроек и команд между iPhone, iPad и Mac
- Облачные изменения сразу доступны на всех устройствах

---

## Профессиональные функции

### SSH-терминал
- Поддержка ключей RSA, ED25519, P256
- Управление вкладками, полная эмуляция терминала
- Поиск с подсветкой и помощь AI

### SFTP-менеджер
- Двухпанельный интерфейс с drag-and-drop
- Операции с файлами, изменение прав, отслеживание прогресса
- Предпросмотр различных типов файлов

### Переадресация портов
- Локальная/удаленная переадресация, SOCKS5 прокси
- SSH-туннели для безопасного доступа к внутренним сетям

### Фрагменты команд
- Сохранение команд для быстрого вызова
- Поддержка переменных для создания личной библиотеки
